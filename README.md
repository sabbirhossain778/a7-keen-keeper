## 📝 Project Overview
In a busy world, it’s easy to lose track of when you last reached out. Keen Keeper acts as a central hub where you can log activities, view friendship analytics, and filter through your interaction history to keep your social life organized.

---

## ✨ Key Features
- **Dynamic Activity Logging:** Log interactions (Calls, Texts, Video calls) instantly using React Context API for real-time updates.
- **Live Interaction Analytics:** Visual breakdown of communication methods using **Recharts** to understand social habits at a glance.
- **Filterable Interaction Timeline:** A history log with detailed dates, searchable and filterable by interaction type.
- **Responsive UI:** Fully optimized for mobile and desktop using **DaisyUI**.

---

## 🚀 Technologies Used
- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS & DaisyUI
- **Data Visualization:** Recharts
- **Icons:** React Icons
- **State Management:** React Context API

---

## 📦 Major Dependencies
To run this project, the following key packages are used:
- `next`: ^14.x
- `react`: ^18.x
- `recharts`: For data visualization
- `react-toastify`: For real-time notifications
- `lucide-react` / `react-icons`: For modern iconography
